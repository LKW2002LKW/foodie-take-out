# 平台端目录架构说明

本文档对应项目：`vue/foodie_plateform`

当前平台端已经完成新结构改造，主分层如下：

- `config`：应用级配置
- `api/request`：请求实例与拦截器
- `api/modules`：业务 API 统一出口
- `stores/modules`：Pinia 状态模块
- `router/modules`：路由模块拆分
- `layouts`：布局层
- `pages`：页面层
- `composables/business`：页面业务逻辑层
- `constants`：常量定义
- `utils`：兼容工具出口

## 根目录文件

- `.gitignore`：Git 忽略规则。
- `README.md`：项目说明文件。
- `index.html`：Vite HTML 模板入口。
- `package-lock.json`：npm 依赖锁定文件。
- `package.json`：项目依赖、脚本命令与构建声明。
- `vite.config.js`：Vite 配置文件，包含别名和开发端口配置。
- `平台端目录架构说明.md`：当前这份平台端目录结构说明文档。

## src 顶层文件

- `src/App.vue`：应用根组件，当前只承接路由视图。
- `src/main.js`：应用启动入口，负责挂载 Vue、Element Plus、Pinia、Router 和全局样式。
- `src/style.css`：全局样式文件。

## src/api

- `src/api/auth.js`：平台登录、退出、管理员信息相关原始 API 实现。
- `src/api/config.js`：系统配置读取与更新相关原始 API 实现。
- `src/api/dashboard.js`：平台首页统计与趋势图相关原始 API 实现。
- `src/api/finance.js`：财务结算、抽成配置、财务总览相关原始 API 实现。
- `src/api/merchant.js`：商户分页、详情、审核、启停用相关原始 API 实现。
- `src/api/notice.js`：公告分页、发布、草稿、删除相关原始 API 实现。
- `src/api/order.js`：订单分页、详情、异常订单相关原始 API 实现。
- `src/api/report.js`：营业额、订单、用户增长、热销商品报表相关原始 API 实现。
- `src/api/review.js`：评价分页、差评预警、删除相关原始 API 实现。
- `src/api/user.js`：用户分页、详情、启用、禁用相关原始 API 实现。

### src/api/modules

- `src/api/modules/auth.js`：认证 API 模块统一出口。
- `src/api/modules/config.js`：系统配置 API 模块统一出口。
- `src/api/modules/dashboard.js`：首页统计 API 模块统一出口。
- `src/api/modules/finance.js`：财务管理 API 模块统一出口。
- `src/api/modules/merchant.js`：商户管理 API 模块统一出口。
- `src/api/modules/notice.js`：公告管理 API 模块统一出口。
- `src/api/modules/order.js`：订单监控 API 模块统一出口。
- `src/api/modules/report.js`：数据报表 API 模块统一出口。
- `src/api/modules/review.js`：评价管理 API 模块统一出口。
- `src/api/modules/user.js`：用户管理 API 模块统一出口。

### src/api/request

- `src/api/request/index.js`：Axios 请求实例统一入口，负责基础配置并挂载拦截器。
- `src/api/request/interceptors.js`：请求/响应拦截器，负责 token 注入、错误提示和登录失效跳转。

## src/composables

- `src/composables/useCharts.js`：首页趋势图表渲染的通用 ECharts 逻辑。
- `src/composables/useMerchant.js`：商户管理基础通用逻辑，负责分页、详情、审核与启停用。

### src/composables/business

- `src/composables/business/useConfigPage.js`：系统配置页业务逻辑，负责配置加载、单项更新、批量保存。
- `src/composables/business/useDashboardPage.js`：首页业务逻辑，负责统计数据、趋势数据和图表刷新。
- `src/composables/business/useFinancePage.js`：财务管理页业务逻辑，负责结算分页、抽成配置和财务概览。
- `src/composables/business/useLoginPage.js`：登录页业务逻辑，负责表单校验、登录提交与登录态同步。
- `src/composables/business/useMerchantPage.js`：商户管理页业务逻辑入口，对 `useMerchant` 做页面层封装。
- `src/composables/business/useNoticePage.js`：公告管理页业务逻辑，负责分页、发布、草稿和删除。
- `src/composables/business/useOrderPage.js`：订单监控页业务逻辑，负责筛选、详情和异常订单查询。
- `src/composables/business/useReportPage.js`：数据报表页业务逻辑，负责图表请求、图表实例和 Tab 切换。
- `src/composables/business/useReviewPage.js`：评价管理页业务逻辑，负责评价列表、差评预警、删除和筛选。
- `src/composables/business/useUserPage.js`：用户管理页业务逻辑，负责分页、详情和启停用。

## src/config

- `src/config/env.js`：环境变量统一读取入口，如平台接口基础地址和请求超时时间。
- `src/config/pinia.js`：Pinia 实例创建与导出入口。
- `src/config/router.js`：路由相关基础配置，如登录页、首页和未登录跳转地址。

## src/constants

- `src/constants/storageKeys.js`：本地存储 key 常量，统一维护 `platform_token` 和 `platform_user`。

## src/layouts

- `src/layouts/MainLayout.vue`：平台后台主布局容器，组合左侧导航和顶部栏。

### src/layouts/components

- `src/layouts/components/HeaderBar.vue`：后台顶部栏组件，负责当前页面标题和退出登录入口。
- `src/layouts/components/SidebarNav.vue`：后台左侧导航组件，负责主菜单展示和路由跳转。

## src/pages

### src/pages/auth/pages

- `src/pages/auth/pages/Login.vue`：管理员登录页视图层，负责展示品牌区和登录表单。

### src/pages/config/pages

- `src/pages/config/pages/SystemConfig.vue`：系统配置页视图层，负责展示参数表和保存入口。

### src/pages/dashboard/pages

- `src/pages/dashboard/pages/Dashboard.vue`：平台首页视图层，展示核心指标卡、汇总栏和经营趋势图。

### src/pages/finance/pages

- `src/pages/finance/pages/FinanceCenter.vue`：财务管理页视图层，包含结算管理、抽成配置和财务报表。

### src/pages/merchant/pages

- `src/pages/merchant/pages/MerchantList.vue`：商户管理页视图层，负责商户列表、详情弹窗和审核弹窗入口。

### src/pages/notice/pages

- `src/pages/notice/pages/NoticeList.vue`：公告管理页视图层，负责公告列表、发布弹窗和草稿保存入口。

### src/pages/order/pages

- `src/pages/order/pages/OrderMonitor.vue`：订单监控页视图层，负责订单筛选、详情弹窗和异常订单弹窗。

### src/pages/report/pages

- `src/pages/report/pages/ReportCenter.vue`：数据报表页视图层，负责营业额、订单、用户增长和热销商品展示。

### src/pages/review/pages

- `src/pages/review/pages/ReviewList.vue`：评价管理页视图层，负责评价列表和差评预警切换。

### src/pages/user/pages

- `src/pages/user/pages/UserList.vue`：用户管理页视图层，负责用户列表、详情弹窗和状态切换。

## src/router

- `src/router/index.js`：路由实例入口，汇总所有路由模块并挂载全局守卫。

### src/router/guards

- `src/router/guards/afterEach.js`：全局后置守卫，负责根据路由标题设置页面标题。
- `src/router/guards/beforeEach.js`：全局前置守卫，负责登录校验和未登录跳转。

### src/router/modules

- `src/router/modules/auth.js`：认证相关路由模块。
- `src/router/modules/config.js`：系统配置相关路由模块。
- `src/router/modules/dashboard.js`：首页相关路由模块。
- `src/router/modules/finance.js`：财务管理相关路由模块。
- `src/router/modules/merchant.js`：商户管理相关路由模块。
- `src/router/modules/notice.js`：公告管理相关路由模块。
- `src/router/modules/order.js`：订单监控相关路由模块。
- `src/router/modules/report.js`：数据报表相关路由模块。
- `src/router/modules/review.js`：评价管理相关路由模块。
- `src/router/modules/user.js`：用户管理相关路由模块。

## src/stores

### src/stores/modules

- `src/stores/modules/dashboardStore.js`：首页统计数据与趋势图数据状态模块。
- `src/stores/modules/platformStore.js`：平台登录态、token、管理员信息状态模块。

## src/utils

- `src/utils/request.js`：请求工具兼容出口，当前统一转发到 `@/api/request`。

## 当前结构说明

- 平台端已经彻底移除旧 `layout` 和 `views` 目录依赖，全部路由入口都在 `src/pages`。
- `src/api/*.js` 保留原始实现，`src/api/modules/*.js` 作为新结构统一出口。
- `src/composables/business` 是页面主要业务层，页面组件尽量只负责展示。
- 当前仍有一个后续优化点：`echarts` 和主入口 chunk 体积较大，后续可继续做 `manualChunks` 拆包优化。
